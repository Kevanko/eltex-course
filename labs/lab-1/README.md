# 1 - Битовые операции

## 1 задание
Вывести двоичное представление целого положительного числа, используя битовые операции.
> (число вводится с клавиатуры).

```bash
$ make run
enter task num (1-4): 1
enter num: 6
result: 110
```

## 2 задание
Вывести двоичное представление целого отрицательного числа, используя битовые операции.
> (число вводится с клавиатуры).

```bash
$ make run
enter task num (1-4): 2
enter -num: -15
result: 11111111111111111111111111110001
```

## 3 задание
Найти количество единиц в двоичном представлении целого положительного числа.
> (число вводится с клавиатуры).

```bash
$ make run
enter task num (1-4): 3
enter num: 256
count <1>: 1
```

## 4 задание
Поменять в целом положительном числе (типа int) значение третьего байта на введенное пользователем число.
> (изначальное число также вводится с клавиатуры).

```bash
$ make run
enter task num (1-4): 4     
enter num: 123456
enter byte [AF, F0, 14]: f0
result: 15786560
before: 00000000000000011110001001000000
after:  00000000111100001110001001000000
```
