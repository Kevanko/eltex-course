# 9 - Файловая система

## Задание
1) Написать программу, которая создаёт файл `output.txt`, записывает в него
строку «String from file», затем считывает её с конца и выводит на экран.

2) Файловый менеджер как mc: две панели, ходим по папкам, Tab меняет панель.
Рисуем через ncurses.

## Структура проекта
```bash
├── bin              # Собранные бинарники (part1, my_mc)
├── include
│   └── my_mc.h      # Структуры, константы, прототипы менеджера
├── Makefile
├── README.md
└── src
    ├── part1.c           # П.1: запись/чтение файла с конца
    ├── my_mc_main.c      # Точка входа, инициализация ncurses
    ├── my_mc_fs.c        # Каталоги: чтение, сортировка, переход по пути
    ├── my_mc_ui.c        # Отрисовка панелей, строки F-клавиш, подсказок
    └── my_mc_input.c     # Главный цикл (клавиатура)
```

## Сборка и запуск
```bash
make                 # part1 и my_mc
make run             # по умолчанию запуск my_mc
make run-my_mc       # только файловый менеджер
make run-part1       # только задание с файлом
make clean           # удалить bin/
```

## Управление (my_mc)
- **Tab** другая панель. **Enter** зайти в папку. **Backspace** вверх по каталогам.
- **F10**, **q** или **Q** выход. Управление только с клавиатуры.

Верхнее меню и строка **F1-F10** только для вида (заглушка, свои подписи), с клавиатуры реально **F10**. Нижняя строка ввода тоже декор, текст копится, ничего не выполняется.
