# 11 - Потоки

## Задание
Реализовать магазин с 5 ларьками, 3 покупателями и 1 погрузчиком.

В каждом ларьке на старте случайное количество товара `900 - 1100`.

У каждого покупателя случайная потребность `9900 - 10100`. Покупатель заходит в случайный свободный ларек, скупает товар до опустошения ларька или пока его потребность не станет `0`. Если не насытился, засыпает на 2 секунды и потом снова идет в ларек.

Погрузчик работает в отдельном потоке. Он выбирает случайный ларек, пополняет его на `200` товаров и засыпает на 1 секунду.

Для каждого ларька используется отдельный мьютекс, поэтому одновременно внутри одного ларька может быть только один поток.

Программа работает, пока все 3 покупателя не насытятся. После этого главный поток завершает погрузчика и завершает программу.

## Структура проекта
```bash
├── bin              # Собранный бинарник
├── Makefile
├── README.md
└── src
    └── main.c       # Магазин, покупатели, погрузчик, мьютексы и логи
```

## Сборка и запуск
```bash
make                 # Сборка
make run             # Сборка и запуск
make clean           # Удаление бинарника
```

## Проверка
> При запуске создаются ларьки, затем стартуют покупатели и погрузчик
```bash
make run

Ларек 1 стартует с 963 товарами
Ларек 2 стартует с 1008 товарами
Ларек 3 стартует с 1095 товарами
Ларек 4 стартует с 918 товарами
Ларек 5 стартует с 1040 товарами
Погрузчик проснулся
Погрузчик пополнил ларек 2 на 200, теперь товаров = 1208
Погрузчик уснул на 1 секунду
Покупатель 1 проснулся, потребность = 10072
Покупатель 1 зашел в ларек 4, товаров = 918
Покупатель 1 купил 918, осталось потребности = 9154
Покупатель 1 уснул на 2 секунды
```

Порядок строк и числа при каждом запуске будут разными, потому что потоки работают параллельно и все значения генерируются случайно.
