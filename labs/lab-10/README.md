# 10 - Процессы

## Задание
1) Реализовать программу, которая порождает процесс. Родительский и дочерний процессы выводят на экран свой `pid`, `ppid`. Затем родительский процесс ждет завершения дочернего и выводит его статус завершения.

2) Реализовать программу, которая порождает `process1` и `process2` и ждет завершения дочерних процессов. `process1` в свою очередь порождает `process3` и `process4` и ждет их завершения. `process2` порождает `process5` и ждет его завершения. Все процессы выводят на экран свой `pid`, `ppid`.

3) Реализовать аналог командного интерпретатора `bash`. При запуске программы пользователь вводит имя программы и опции запуска. Программа порождает процесс и в нем выполняет введенную программу с заданными опциями, ждет завершения дочернего процесса и снова возвращается к вводу следующей команды. Выход по команде `exit`.

## Структура проекта
```bash
├── bin              # Собранные бинарники
├── Makefile
├── README.md
└── src
    ├── 1.c          # Порождение одного дочернего процесса, wait, статус
    ├── 2.c          # Дерево процессов: process1-process5
    └── 3.c          # Простой интерпретатор команд через fork/exec
```

## Сборка и запуск
```bash
make                 # Сборка всех заданий
make run-1           # Запуск 1 задания
make run-2           # Запуск 2 задания
make run-3           # Запуск 3 задания
make clean           # Удаление бинарников
```
